## 三、设计系统全面升级（7条设计规范落地）

### 1️⃣ 清晰文本对齐与换行
```css
--line-height-tight: 1.25;    /* 标题 */
--line-height-base: 1.50;     /* 正文 */
--line-height-relaxed: 1.75;  /* 段落 */
```
- 所有文本统一基线对齐
- overflow-wrap: anywhere 处理长内容
- 合理换行提升可读性

### 2️⃣ 权重区分拉开对比
```css
--font-weight-normal:   400;  /* 正文内容 */
--font-weight-medium:   500;  /* 辅助信息 */
--font-weight-semibold: 600;  /* 次要标题 */
--font-weight-bold:     700;  /* 主要标题 */
```
- 标题使用bold(700)，正文使用normal(400)
- 字重差异明显，引导用户视线
- letter-spacing微调提升质感

### 3️⃣ 主次操作明确区分
**主按钮**:
- 背景色：var(--accent)
- hover: 变深 + 上浮1px + 阴影加深
- active: 回归原位

**次按钮(ghost)**:
- 描边样式，白色背景
- 文字色：var(--accent)
- 无hover上浮效果

**统一过渡时间**: transition: 150ms ease

### 4️⃣ 圆角半径准确嵌套
```css
--radius-sm: 6px;   /* 输入框、小卡片、标签 */
--radius-md: 8px;   /* 面板、弹窗、中等容器 */
--radius-lg: 12px;  /* 大容器、页面级区块 */
--radius-full: 999px; /* 药丸形、圆形元素 */
```
- 内外圆角协调统一
- 小元素用小圆角，大容器用大圆角
- 视觉更舒适自然

### 5️⃣ 柔和阴影透气自然
```css
--shadow-sm: 0 1px 3px rgba(20,103,179,0.04);
--shadow-md: 0 4px 12px rgba(20,103,179,0.06);
--shadow-lg: 0 8px 24px rgba(20,103,179,0.08);
```
- 使用环境色投影（#1467b3），而非黑色
- 透明度控制在4%-8%，干净不脏
- 三层阴影层级适应不同场景

### 6️⃣ 统一间距规范布局
```css
--space-xs:  4px;    /* 元素内部极小间距 */
--space-sm:  8px;    /* 紧凑间距、组件内 */
--space-md:  12px;   /* 标准间距 */
--space-base:16px;   /* 基础间距单元 */
--space-lg:  24px;   /* 宽松间距、区块间 */
--space-xl:  32px;   /* 大间距、章节分隔 */
```
- 基于4px倍数体系
- 保持间距一致，页面规整有序
- 视觉节奏清晰

### 7️⃣ 更清晰的突出内容展示
**策略卡片优化**:
- hover时：边框变蓝(#b8d4f0) + 轻微上浮(-2px)
- 标题加粗(bold) + 紧凑行高(tight)
- 低调描边标签不抢视线
- 内容区左侧竖线引导阅读

**信息卡片优化**:
- hover时阴影加深 + 边框变蓝
- 标题与内容字重对比明显
- 字段列表标签/值区分清晰

**司机卡片优化**:
- hover时背景变色 + 阴影浮现
- ID号加粗显示
- 辅助信息弱化处理

## 四、新增Design Token系统

在`:root`中定义完整的设计变量体系：
- **颜色Token**: --bg, --panel, --line, --text, --muted, --accent等
- **圆角Token**: --radius-sm/md/lg/full
- **间距Token**: --space-xs/sm/md/base/lg/xl
- **阴影Token**: --shadow-sm/md/lg
- **字重Token**: --font-weight-normal/medium/semibold/bold
- **行高Token**: --line-height-tight/base/relaxed
